# Claude Analysis Prompts — Diary & TGR

วิธีใช้: อัปโหลดไฟล์ `diary-claude-YYYY-MM-DD.md` และ/หรือ `tgr-claude-YYYY-MM-DD.md` เข้า Claude แล้ว copy prompt ด้านล่างไปใช้ได้เลย

## 📔 Diary Analysis

### สรุปภาพรวม
```
อ่านไฟล์ diary นี้แล้วสรุปให้ฉัน:
1. แนวโน้มอารมณ์ช่วง 30 วันที่ผ่านมา — บวกหรือลบมากกว่ากัน?
2. นิสัยไหนทำได้สม่ำเสมอที่สุด และไหนที่ขาดบ่อย?
3. สิ่งที่ฉันขอบคุณบ่อยที่สุด top 5
4. ตัวตนที่ฉันโหวตให้บ่อยที่สุดคืออะไร?
5. จุดที่ควรปรับปรุงมากที่สุด 1 อย่าง
```

### วิเคราะห์เชิงลึก
```
จากข้อมูล diary นี้ ช่วยวิเคราะห์:
- วันไหนบ้างที่ฉันรู้สึกดีที่สุด มีอะไรเหมือนกัน?
- วันไหนบ้างที่รู้สึกแย่ เกิดจากอะไร?
- แผนที่ตั้งไว้ vs ทำได้จริง คิดเป็นกี่เปอร์เซ็นต์?
- แนะนำ 3 อย่างที่ฉันควรทำต่อเนื่องและ 3 อย่างที่ควรหยุด
```

### รายสัปดาห์
```
สรุปสัปดาห์นี้จาก diary ให้หน่อย:
- ไฮไลต์ที่ดีที่สุด
- ความท้าทายที่เจอ
- แผนสัปดาห์หน้าที่แนะนำ
```

## 📈 TGR Dashboard Analysis

### สรุปภาพรวม
```
อ่านข้อมูล TGR นี้แล้วสรุป:
1. หลักการ 13 ข้อไหนที่ฉันทำได้ดีที่สุดและแย่ที่สุด?
2. พิธีกรรมเช้า-เย็น ทำได้กี่เปอร์เซ็นต์เฉลี่ย?
3. ผีแห่งความกลัวไหนโผล่บ่อยที่สุด?
4. แนวโน้มคะแนนรายสัปดาห์ดีขึ้นหรือแย่ลง?
```

### โฟกัสหลักการ
```
จาก TGR data นี้ หลักการไหนที่ฉันคะแนนต่ำสุดติดต่อกัน?
ช่วยอธิบายว่าถ้าฉันปรับปรุงหลักการนั้น จะส่งผลต่อเป้าหมาย [ระบุเป้าหมาย] อย่างไร
และแนะนำ action ที่ทำได้จริงใน 7 วันข้างหน้า
```

## 🔗 วิเคราะห์รวม Diary + TGR

```
ฉันอัปโหลดทั้ง diary และ TGR data มาให้ ช่วยวิเคราะห์ภาพรวมชีวิต:

1. วันที่พิธีกรรม TGR ครบ — diary บันทึกอารมณ์ดีขึ้นไหม?
2. หลักการ TGR ที่คะแนนต่ำ — ส่งผลกับนิสัยรายวันใน diary ยังไง?
3. ภาพรวม: ฉันกำลังเดินไปในทิศทางที่ถูกต้องไหม?
4. สิ่งที่ควรโฟกัสมากที่สุดในเดือนหน้าคืออะไร?
```

## 🔄 วิธี Export ไฟล์

1. เปิด **diary.html** หรือ **TGR-Dashboard.html** ในเบราว์เซอร์
2. ไปที่แท็บ **📊 สถิติ** (diary) หรือ **📈 พัฒนาการ** (TGR)
3. กดปุ่ม **📋 Export .md สำหรับ Claude**
4. ไฟล์จะดาวน์โหลดลงเครื่อง → ย้ายมาไว้ใน folder นี้
5. แนบไฟล์ + copy prompt ด้านบน → วาง Claude ได้เลย
